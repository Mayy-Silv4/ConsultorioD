# 🦷 Projeto Sorriso Perfeito - Landing Page

## 🦷 Sobre o Projeto

A Sorriso Perfeito é uma clínica odontológica infantil fictícia desenvolvida como projeto acadêmico com o objetivo de aplicar conhecimentos em HTML5, CSS3 e JavaScript na criação de uma landing page profissional, moderna e responsiva.

O site foi desenvolvido com foco em:

Experiência do usuário (UX)

Design acolhedor e infantil

Navegação fluida

Responsividade para desktop e mobile

Integração com WhatsApp para agendamentos

## 🎯 Objetivo do Projeto

Criar uma landing page para um consultório odontológico infantil moderno contendo:

Apresentação da clínica

Seção de serviços

Galeria de pacientes

Depoimentos

Formulário de contato integrado ao WhatsApp

Layout responsivo

Identidade visual própria (paleta + tipografia)

## 🌐 Visão Geral do Site

**🏠 Home**

Apresentação principal da clínica com chamada de ação para agendamento e imagem ilustrativa da dentista.

**🦷 Serviços**

Exibição dos principais serviços oferecidos:

Limpeza e prevenção

Tratamento ortodôntico

Clareamento infantil

Layout estruturado com Flexbox para alinhamento responsivo.

**📸 Galeria de Sorrisos**

Seção com fotos circulares dos pacientes e frases personalizadas abaixo de cada imagem.

**💬 Depoimentos**

Comentários fictícios de pacientes destacando a experiência positiva no atendimento.

**📞 Contato**

Formulário funcional que redireciona automaticamente para o WhatsApp com mensagem personalizada.

**📍 Footer**

Rodapé com identidade visual da clínica, navegação rápida e direitos reservados.

## 🚀 Como Executar o Projeto

Clone o repositório:

git clone https://github.com/seu_usuario/projeto-sorriso-perfeito.git

Acesse a pasta:

cd projeto-sorriso-perfeito

Abra o arquivo index.html no navegador.

💡 Dica: Utilize a extensão Live Server no VS Code para melhor visualização.
